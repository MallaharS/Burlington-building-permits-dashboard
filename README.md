# Burlington Building Permits Dashboard

A Power BI dashboard I built to explore building permit data from the City of Burlington's open data portal. It looks at where and when construction is happening across the city, and what kind of work is being done.

## What's in it

- Key numbers up top: total permits, total construction value, average value, and active vs. closed permits
- Permits and construction value over time
- A breakdown of permits by type
- Permits and construction value by ward
- Filters for year, permit type, ward, and status so you can drill into the data

## Data

City of Burlington open data — building permits (CSV), around 5,000 permits.

## Built with

Microsoft Power BI

## Notes

The current year only has partial data, so its totals look lower. "Active" permits are ones marked Issued; "closed" ones are marked Final.
